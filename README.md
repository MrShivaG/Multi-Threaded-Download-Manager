# ⚡ Multi-Threaded Download Manager (Java CLI)

A **CLI-based multi-threaded file downloader** written in Java.  
It improves download speed by downloading files simultaneusly in  using threads.

---

## 🚀 Features

- Multi-threaded downloading (parallel downloads)
- CLI-based lightweight tool
- Efficient handling of large files
  

---

## 🧠 Working Principle

1. Fetch file size from server
2. Download each file in diffrent thread
---

## 📦 Requirements

- Java 8+

---

## 🛠 Compilation

```bash
javac Main.java
